# Cityscapes-Semantic-Segmentation-with-FCN-DeepLabV3-PyTorch-
End-to-end semantic segmentation on the Cityscapes dataset using PyTorch. Implements custom dataset wrapping, label remapping, training and evaluation of FCN-ResNet50 and DeepLabV3-ResNet50, with loss handling, confusion matrices, mIoU metrics, and qualitative visual comparisons.
